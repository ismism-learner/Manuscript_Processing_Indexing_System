# 哲学文稿处理与索引系统

## 项目简介

**哲学文稿处理与索引系统**是一个为文科知识分子、研究者和学生设计的强大工具。它旨在自动从哲学文稿中提取核心概念与次级概念，并以结构化的方式呈现它们的定义、解释和例子。本系统通过调用先进的AI模型（基于硅基流动API）进行深度文本分析，帮助用户快速梳理复杂的哲学思想，构建知识图谱。

本工具最大的特点是其易用性——它是一个无需任何安装和配置的独立网页应用，用户只需在浏览器中打开即可开始使用。

## 核心功能

*   **概念提取**: 自动识别并提取文本中的核心哲学概念及其下属的次级概念。
*   **结构化分析**: 为每个概念提供清晰的定义、详尽的说明和具体的例子。
*   **“四论域”并行处理**: 独创的多维度分析框架，从不同哲学层面深度解读文本。
*   **无需安装**: 作为一个独立的 `standalone.html` 文件，无需复杂的环境配置，开箱即用。
*   **多格式支持**: 支持 `.txt`, `.docx`, `.md` 多种主流文稿格式。
*   **Markdown输出**: 生成结构清晰、排版精美的 Markdown 格式分析报告，便于后续编辑、分享和存档。

## 独特优势：“四论域”架构

本系统的核心创新在于其独特的“四论域”并行处理架构。它并非简单地对文本进行线性分析，而是将文稿中的概念分发到四个预设的哲学论域中，进行同步、并行的深度处理。

这四个论域分别是：

1.  **场域主体对于世界的幻想形式本体**: 分析主体如何构建其精神世界的想象。
2.  **主体对于世界当中存在的实际的物质进行一个分析**: 探讨主体如何理解和分析现实物质世界。
3.  **实用主义方面的认识论**: 从认识论的角度，揭示用户主体的意识形态。
4.  **实用主义方面的目的论**: 从目的论的角度，判断用户主体的意识形态走向。

通过这种多维度、同步进行的分析，本工具能够揭示不同哲学领域之间的隐藏联系，为用户提供一个远比传统工具更全面、更立体、更深刻的哲学文本解读。

## 工作流程

1.  **输入**: 用户将 Word 文稿（需先转换为 `.docx` 格式）、Markdown 文件或纯文本文件上传或拖拽到应用界面。
2.  **处理**: 应用内置的 JavaScript 代码调用用户配置的“硅基流动”API，对文本进行多次AI处理。
    *   **第一次处理**: 提取核心概念。
    *   **第二次处理**: 为每个核心概念生成定义和解释。
    *   **第三次处理**: 寻找相关的例子和次级概念。
3.  **输出**: 系统整合所有分析结果，生成一份结构化的 Markdown (`.md`) 文件，用户可直接下载保存。

## 快速上手指南

该工具是一个独立的HTML文件，无需安装任何依赖，可以直接在现代浏览器（推荐使用Chrome或Edge）中打开并运行。

1.  **获取API密钥**:
    *   访问 [硅基流动官方网站](https://www.siliconflow.cn/) 并注册一个账户。
    *   登录后，在您的账户设置中申请并复制您的API密钥。

2.  **打开工具**:
    *   下载本仓库中的 `standalone.html` 文件到您的本地电脑。
    *   在您的浏览器中直接打开这个HTML文件。

3.  **配置API密钥**:
    *   在页面右上角的“设置”区域，将您复制的API密钥粘贴到输入框中。
    *   系统将自动连接并验证您的密钥。

4.  **上传文稿**:
    *   将您的 `.txt`, `.docx`, 或 `.md` 格式的文稿文件拖拽到指定区域，或点击上传区域选择文件。
    *   工具会自动读取文件内容并准备进行分析。
